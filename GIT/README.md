# GIT COMMANDS :

**GIT INIT** 
     TO INTIALIZE GIT

**GIT ADD**
     TO TRACK THE FILE

    GIT ADD * 
        TO TRACK ONLY FILES AND NOT HIDDEN FILES

    GIT ADD . 
        TO TRACK ALL THE HIDDEN FILES TOO
        
**GIT COMMIT -M "COMMIT_MESSAGE"**
     TO MAINTAIN THE VERSIONS OF THE FILE
